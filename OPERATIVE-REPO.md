# Operative Governance Repository

The operative governance documents of Radix DLT DAO LLC — the documents currently in force and the proposals under active consideration — are maintained in a separate repository:

**[https://github.com/Shadaffy/radix-dao-governance](https://github.com/Shadaffy/radix-dao-governance)**

That repository is updated only when a governance proposal passes. If it's there, it's law.

---

## About this repository

This is the **reference library** — the complete governance framework including:

- The full policy library (all phases, including policies not yet activated)
- Working drafts and proposal templates
- Activation roadmap and transition framework
- Legal formation document history and templates
- Impact analysis and supporting materials

This repository is the source from which operative documents are drawn. It is intended for governance architects, working group members, RAC members, and anyone doing impact analysis on proposed changes.

If you are a token holder, community member, or external party looking for what currently governs the DAO, go to the operative repository linked above.
